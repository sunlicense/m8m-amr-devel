SiPS

( STARWING intelligent indoor
positioning system )

SiPS is the most advanced indoor positioning solution in the world. It combines the unique AOA(Angle-of-Arrival) signal tracking device with A.I. computing and analysis technology. With AOA device, the positioning accuracy reaches centimeter-level, 10 times superior than industry average. By adopting A.I. technology of machine learning and neural network, SiPS can explore the insightful information behind those location data, such as historical path tracing, path prediction, behavior analysis, heatmap analysis, and friendship map. SiPS’s total solution includes hardware, software, interface customization, system integration and cloud service. SiPS has assisted numerous customers in smart field upgrade, such as. equipment automatize, asset & personnel tracking, behavior analysis, security surveillance.
